SOP of creating a product

Database part
Create product model and migration
Add relation in Product.php
Add necessary code in specific product model
Add table in specific product migration

File part
Create product blade

API part
Add api link in api.php
Add api for specific product in ProductController.php
Update api for all product in ProductController.php

Route part
Add route in web.php
Add show product function in ProductController.php
Update the switch loop of show all product function in ProductController.php
Update nav link

Function part
Add store function for specific product in ProductController.php
Update the switch loop in store function for all product in ProductController.php 
Add update function for specific product in ProductController.php
Update the edit function to show the right form
Update the switch loop in update function for all product in ProductController.php
